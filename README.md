To use this program, copy the encounter data for a single location—starting from Land and ending with the final Fishing encounter—and paste it into a file named "base.txt" located in the same directory as PorygonZ.

Here’s an example of how the content in the file should look:
````
Land,6
    10,KANGASKHAN,55,58
    10,PIDGEOT,55,58
    10,SUNFLORA,55,58
    10,HERACROSS,55,58
    10,TINKATON,55,58
    10,LILLIGANT,55,58
    10,FLORGES,55,58
    10,BEWEAR,55,58
    5,ESPATHRA,55,58
    5,APPLIN,55,58
    5,CHARMELEON,35,40
    5,IVYSAUR,35,40
OldRod
    50,JELLICENT,55,58
    25,AZUMARILL,55,58
    25,SEAKING,55,58
GoodRod
    20,BASCULIN,55,58
    45,AZUMARILL,55,58
    25,CRAWDAUNT,55,58
SuperRod
    30,WARTORTLE,35,40
    70,VELUZA,55,58
````
